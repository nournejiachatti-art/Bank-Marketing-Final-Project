# Projet Classification Bancaire - ML Avancé

Ce projet implémente une solution complète de classification pour prédire les souscriptions aux dépôts à terme.

## 🚀 Tâches Réalisées
- **Modèles implémentés :** KNN, SVM, Random Forest, Logistic Regression.
- **Périmètre Avancé :** Optimisation des hyperparamètres (Depth, Neighbors) via interface.
- **Tracking MLOps :** Utilisation de MLflow pour le suivi des métriques (Accuracy, F1, Precision, Recall).
